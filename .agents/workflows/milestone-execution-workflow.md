---
description: Milestone execution workflow
---

1. Read PROJECT_CHARTER.md, CONTRACTS.md, MILESTONES.md, ACCEPTANCE_CHECKLIST.md, and COMPATIBILITY_MATRIX.md.
2. Identify the exact milestone/sub-milestone in scope.
3. Produce a task list bounded strictly to that scope.
4. Produce an implementation plan naming the exact crates/files to modify.
5. State what is explicitly out of scope.
6. Implement only after the plan is reviewed.
7. Add or update tests.
8. Run validation commands.
9. Produce a walkthrough that includes exact evidence and unresolved limitations.
10. Do not claim milestone completion without matching acceptance evidence.
